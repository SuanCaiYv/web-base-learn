现在来试试多列布局，即在页面上划分出许多列，像报纸那样。

同样就像开启弹性盒子一样，我们在多列的父元素中使用column-count/column-width开启多列布局

在这里我们需要注意的是，column-width属性不会完全产生指定大小的列，而是会略大于这个值，假设值为x，页面总宽度为y，列数为k，则有k = (y/x)取整；实际列宽=y/k >= x

此外，想要实现列不被折断，即一直显示下去，可以使用page-break-inside属性并设置值为avoid

此外还有设置列的间隙，列与列之间的分割线样式之类的，参见CSS。

记二

这首歌是中秋后，我和我弟去医院看我爸，回来的路上，他的手机蓝牙放的。
车上就我们俩，我们半开着车窗，太阳还没完全落下，空气依然保留着足够的热度，任由这样的风灌进衣袖；为了中和风声，音乐开得很大声，看起来就是两个人在一条美丽但是人迹罕至的省道上飙着车。
两个人都刚失恋，都在假装自己不在乎，都在劝对方走出来，因为他要高考，我要秋招。中秋节那晚，两个人很豪气，吃了一百多的烤串，喝了点酒，看着被遮住的月亮慢慢溜出来。然而再也没人祝福，也没人期待着去团圆。他爸在打工，我爸在躺着，母亲们早早地睡去。那晚说了很多，但是我记不得很多，我是个讲感受的人，所以那晚我感受了很多，心里不停地在转，在想，第一次感觉到百感交集。因为失望，悲伤，无力，害怕，很多感觉全部涌上来了，我不知道该先处理哪个。
可能吧，我说可能，就在那一刻，我突然感觉和曾经的人有了隔阂感，和那个学校，里面的人，那座城市，有了一种强烈的抗拒和想要疏远的感觉。后来我选择去另一个陌生的城市，就像我三年前逃离那里一样。
